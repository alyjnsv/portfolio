🚀 **Live site:** [https://alyjnsv.github.io](https://alyjnsv.github.io)

Персональный сайт-портфолио **AI Architect & Automation Engineer**.

## Технологии сайта

- **Pure HTML/CSS/JS** — без фреймворков, максимальная скорость
- **Дизайн** — вдохновлен [Firecrawl.dev](https://firecrawl.dev) (dark theme, orange #fa5d19)
- **Анимации** — SVG-анимация n8n workflow с движущимися data packets
- **Хостинг** — GitHub Pages

## Особенности

- 🎨 Тёмная тема с фирменным оранжевым акцентом (`#fa5d19`)
- ⚡ Animated n8n workflow diagram с реальными нодами
- 📊 Анимированные счётчики статистики
- 🖱️ Cursor glow эффект
- 🃏 3D tilt эффект на карточках проектов
- 📱 Полностью адаптивный дизайн
- 🔍 SEO-оптимизированный

## Запуск локально

```bash
# Python (любая версия)
python3 -m http.server 8080
# Открыть: http://localhost:8080
```
